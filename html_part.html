#Examples
index

{ Tag }

<html> ~ </html>

  • 웹페이지의 시작과 끝

<head> ~ </head>

  • 웹페이지의 정보를 정의

<title> ~ </title>

  • 문서의 제목을 정의
  • 정의한 제목은 브라우저의 제목표시줄 또는 페이지 탭에 표시

<body> ~ </body>

  • 웹페이지의 본문을 출력
  • HTML 문서에는 하나의 body 태그만 존재 가능

<h1~h6> ~ </h1~h6>

  • 문서의 제목을 정의
  • 숫자가 낮을수록 중요한 제목임을 의미

<p> ~ </p>

  • 하나의 문단을 출력
  • 위, 아래에 빈 줄이 함께 출력됨

<b> ~ </b>

  • 굵은 텍스트를 지정

<br> ~ </br>

  • 줄 바꿈 태그

<ol> ~ </ol>

  • 순서가 있는 목록을 출력
  • 한 개 이상의 li 태그가 필요

<ul> ~ </ul>

  • 순서가 없는 목록을 출력
  • 한 개 이상의 li 태그가 필요

<a> ~ </a>

  • 다른 문서로 이동하는 텍스트를 출력

<img>

  • 이미지를 출력
  • 이미지 속성
    • width/height : 이미지의 크기를 설정
    • alt : 이미지를 가져오지 못했을 때 대신 나타날 문구를 설정
    • src : 이미지 파일의 위치를 설정
    • border : 이미지의 테두리를 설정

<figure> ~ </figure>

  • 컨텐츠에 설명을 추가
  • <img> 태그와 같이 많이 씀

<figcaption> ~ </figcaption>

  • <figure> 태그의 하위 태그로 이 태그에 표시 될 문구를 적음

<audio> ~ </audio>

  • 오디오를 재생
  • 출력과 재생 형식 설정 속성이 존재
    • <source> ~ </source> : 해당 태그로 오디오 파일의 위치와 타입을 설정
    • controls : 오디오 컨트롤러를 표시
    • autoplay : 페이지가 출력되면 오디오가 자동으로 재생
    • loop : 오디오의 반복 재생 여부를 설정
    • preload : 오디오가 재생 되기 전에 파일을 불러오는지의 여부

<video> ~ </video>

  • 비디오를 재생
  • 출력과 재생 형식 설정 속성이 존재
    • 뭐시기
    • 뭐시기

<table> ~ </table>

  • 테이블(표)를 출력

<tr> ~ </tr>

  • 테이블의 행을 정의

<th> ~ </th>

  • 테이블의 제목을 정의(열)

<td> ~ </td>

  • 테이블의 데이터를 정의(열)

<caption> ~ </caption>

  • 테이블의 제목을 정의

<div> ~ </div>

  • 문서의 전체 공간에 대해 분할
  • block 형식으로 분할

<span> ~ </span>

  • 문서의 일부 영역만 분할
  • inline 형식으로 분할

<form> ~ </form>

  • 입력 양식을 생성하는 태그
  • 입력 양식 속성
    • action : 입력값을 넘겨줄 파일을 설정(JSP, PHP 등)
    • method : 웹 서버와 클라이언트 간 통신 방 지정

<select> ~ </select>

  • 선택 목록을 추가하는 태그
  • 선택 목록 속성 & 하위 태그
    • name : 선택 목록의 이름을 설정
    • multiple : 다중 선택 허용 설정
    • <option> ~ </option> : 선택 항목 추가
    • <optgroup> ~ </optgroup> : 선택 항목들을 묶음
      • label 태그로 항목 묶음의 이름을 설정 가능
    • <datalist> ~ </datalist> : 자동완성이 가능한 선택 항목들을 설정

<input> ~ </input>

  • <form> 태그의 하위 태그 폼 양식을 추가
  • 입력 속성
    • value : <input> 태그의 초기값
    • name : <input>를 구별하기 위한 이름
    • onclick : 버튼 클릭 시 실행 될 JS 메서드를 지정
      • ex) onclick="alert('Clicked!')"
    • type : <input> 태그의 유형을 지정
      • text : 텍스트를 입력 받음
      • password : 비밀번호를 입력 받음. 입력값은 점으로 대체되어 출력
      • radio : 중복 선택이 불가능한 둥근 모양의 선택 버튼
      • checkbox : 중복 선택이 가능한 각진 모양의 선택 버튼
      • file : 드라이브의 파일을 선택할 수 있는 창을 여는 버튼
      • image : 이미지를 버튼으로 만듬. 기능은 submit과 동일
      • hidden : 보이지는 않지만 서버에 전달 되는 버튼
      • submit : 입력값을 전부 서버에 전달하는 버튼
      • button : 평범한 버튼
      • reset : 입력값을 전부 초기화하는 버튼
      • date : 년도를 선택할 수 있는 달력을 출력
      • month : 년-월 형식으로 날짜 입력
      • week : 년-주 형식으로 날짜 입력
      • time : 시간만 입력
      • datetime-local : 년-월-일-시간 형식으로 입력
      • color : 색을 선택할 수 있는 팔레트를 출력
      • number : 숫자를 입력 할 수 있는 양식을 추가
      • email : 이메일 형식에 맞춰서만 입력을 받음
      • url : URL 형식에 맞춰서만 입력을 받음

<fieldset> ~ </fieldset>

  • 입력 양식이 여러개일 경우 하나로 묶고 테두리를 생성
  • <legend> ~ </legend> : fieldset에 출력 될 이름을 설정
    • align : 정렬 방향을 지정. 'align=center'로 중앙 정렬 가능

<meter> ~ </meter>

  • 지정된 값을 막대그래프로 출력
  • <legend> ~ </legend> : fieldset에 출력 될 이름을 설정
    • align : 정렬 방향을 지정. 'align=center'로 중앙 정렬 가능

<progress> ~ </progress>

  • 현재 진행율 막대 그래프로 출력
  • 막대그래프 공통 속성
    • value : 실제로 측정 된 값
    • min, max : 입력 값의 범위(기본값은 0~1)
    • low, high : 허용 되는 입력 값의 범위(min, max 내에 존재하는 범위여야 함)
    • optimum : 데이터가 가지길 원하 기댓값

<marquee> ~ </marquee>

  • 움직이는 콘텐츠를 출력
  • 이동 방식 설정 속성이 존재
    • behavior : scroll | slide | alternate
    • direction : left | right | up | down
    • height : px
    • width : px
    • loop : numeric
    • scrollamount : numeric
