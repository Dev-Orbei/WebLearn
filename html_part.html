{ Tag }

<html> ~ </html>

  • 웹페이지의 시작과 끝

<head> ~ </head>

  • 웹페이지의 정보를 정의

<title> ~ </title>

  • 문서의 제목을 정의
  • 정의한 제목은 브라우저의 제목표시줄 또는 페이지 탭에 표시

<body> ~ </body>

  • 웹페이지의 본문을 출력
  • HTML 문서에는 하나의 body 태그만 존재 가능

<h1~h6> ~ </h1~h6>

  • 문서의 제목을 정의
  • 숫자가 낮을수록 중요한 제목임을 의미

<p> ~ </p>

  • 하나의 문단을 출력
  • 위, 아래에 빈 줄이 함께 출력됨

<b> ~ </b>

  • 굵은 텍스트를 지정

<br> ~ </br>

  • 줄 바꿈 태그

<ol> ~ </ol>

  • 순서가 있는 목록을 출력
  • 한 개 이상의 li 태그가 필요

<ul> ~ </ul>

  • 순서가 없는 목록을 출력
  • 한 개 이상의 li 태그가 필요

<a> ~ </a>

  • 다른 문서로 이동하는 텍스트를 출력

<img>

  • 이미지를 출력

<audio> ~ </audio>

  • 오디오를 재생
  • 출력과 재생 형식 설정 속성이 존재
    • 뭐시기
    • 뭐시기

<video> ~ </video>

  • 비디오를 재생
  • 출력과 재생 형식 설정 속성이 존재
    • 뭐시기
    • 뭐시기

<table> ~ </table>

  • 테이블(표)를 출력

<tr> ~ </tr>

  • 테이블의 행을 정의

<th> ~ </th>

  • 테이블의 제목을 정의(열)

<td> ~ </td>

  • 테이블의 데이터를 정의(열)

<caption> ~ </caption>

  • 테이블의 제목을 정의

<div> ~ </div>

  • 문서의 전체 공간에 대해 분할
  • block 형식으로 분할

<span> ~ </span>

  • 문서의 일부 영역만 분할
  • inline 형식으로 분할

<marquee> ~ </marquee>

  • 움직이는 콘텐츠를 출력
  • 이동 방식 설정 속성이 존재
    • behavior : scroll | slide | alternate
    • direction : left | right | up | down
    • height : px
    • width : px
    • loop : numeric
    • scrollamount : numeric
