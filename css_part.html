{ Attribute }

[Common]

  • height : 높이
  • width : 너비
  • padding : 내부 여백(10px:top 10px:right 5px:bottom 5px:left)
  • margin : 외부 여백(10px:top,bottom 5px:left,right)
  • padding-top/bottom/left/right : 내부 여백을 따로 설정
  • margin-top/bottom/left/right : 외부 여백을 따로 설정
  • box-shadow : 사각형 그림자 생성(10px:x-offset 10px:y-offset 5px:blur black:color)
  • float : 요소를 떠 있게 만듬(레이아웃에서 벗어나게 만듬)
  • clear : float 효과를 제거
  • overflow : visible | hidden | scroll | auto
  • opacity : 투명도
  • transition (property) (duration) (timingfunction) (delay): (property) 속성을 (delay) 후에 (duration)에 걸쳐 (timingfunction) 방식으로 변화시킵니다.

[Font]

  • color : 글자 색상을 설정
  • font : 여러 폰트 설정을 한 번에 설정
  • font-size : 폰트 크기
  • font-family : 폰트 이름
  • font-style : 폰트 기울임 효과(italic)
  • font-weight : 폰트 굵기 효과(normal | bold | bolder | lighter | number | initial | inherit)
  • text-decoration : 폰트 밑줄 효과(line-through | overline | underline) and (solid | double | dotted | dashed | wavy)
  • text-shadow : 글자 그림자(5px:x-offset 5px:y-offset 10px:blur red:color)
  • text-align : 글자 정렬 방식(start | end | center | justify)

[Background]

  • background : 배경 속성을 한번에 설정
  • background-color : 배경 색상
  • background-size : 배경 크기
  • background-image : 배경 이미지 설정 ( url("/img.png") )
  • background-position : top | bottom | left | right | center
  • background-repeat : repeat | repeat-x | repeat-y | no-repeat
  • background-attachment : scroll | fixed | local

[Border]

  • border : 테두리 속성을 한번에 설정 (5px:top 5px:right,left 10px:bottom solid:style black:color)
  • border-top/bottom/left/right : 해당 변의 속성을 한번에 설정
  • border-top/bottom/left/right-color : 해당 변의 색상을 설정
  • border-top/bottom/left/right-style : dotted | dashed | solid | double | groove | ridge | inset | outset | none | hidden
  • border-top/bottom/left/right-width : 해당 변의 굵기를 설정
  • border-top/bottom/left/right-radius : 모서리의 굴곡을 설정 (5px:top-left 5px:top-right,bottom-left 3px:bottom-right - *radius 설정은 대각선으로 연결)

{ Selector }

  • * { } : 전체 선택자

  • tag_name { } : 태그 선택자

  • .class_name { } : 클래스 선택자

  • #id_name { } : 아이디 선택자

  • [attribute] { } : 속성 선택자

    • [attribute=value] : 정확히 value인 모든 속성
    • [attribute~=value] : 가진 값 중에 value가 있는 모든 속성
    • [attribute|=value] : 정확히 value거나 value로 시작하면서 뒤에 '-'가 붙는 모든 속성
    • [attribute^=value] : value를 접두사로 가지고 있는 모든 속성
    • [attribute$=value] : value를 접미사로 가지고 있는 모든 속성
    • [attribute*=value] : value라는 단어를 하나 이상 포함한 모든 속성

  • tag_name1 tag_name2 { } : 조합 선택자

    • tag_name1 tag_name2 : tag_name1 내의 모든 tag_name2 (하위 선택자)
    • tag_name1 > tag_name2 : 부모가 tag_name1인 모든 tag_name2 (자식 선택자)
    • tag_name1 + tag_name2 : tag_name1 바로 뒤에 있는 모든 tag_name2 (인접 형제 선택자)
    • tag_name1 ~ tag_name2 : tag_name1이 바로 앞에 있는 모든 tag_name2 (일반 형제 선택자)

  • :num-child { } : 구조 선택자

    • :first-child : 첫번째 자식 요소 선택
    • :last-child : 마지막 자식 요소 선택
    • :nth-child(m) : m번째 요소 선택 (m에는 2n-1 같은 형태의 수식이 올 수 있음)
    • :nth-last-child(m) : 뒤에서부터 m번째 요소 선택

  • :link|visited|hover|active { } : 가상 선택자

    • link : 방문한 적 없는 링크
    • visited : 방문한 링크
    • hover : 커서를 올렸을 때
    • active : 클릭 또는 선택 했을 때
